Grammar

Grammar mostly follows English, so default to those rules if no rules are specified here

“Ing” is replaced with “met” when the word ends with a vowel and “et” otherwise. When using the altenshard laugiuage the symbol is still the ing symbol

Plural: add “s” (could change) to the end of the word, or when writing in Altenshardian, use the plural simbol

Past tense: add “ed” (could change) to the end of the word, or when writing in Altenshardian use the symbol for past tense

Present tense is just the normal word

Future tense: add “ad” (could change) to the end of the word, or when writing in Altenshardian use the symbol for Future tense
